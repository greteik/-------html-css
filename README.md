# Wishbone+Partners

Лендинг архитектурного бюро с адаптивом и интерактивными элементами.

## Функционал
✅ Адаптивная верстка (Mobile First)  
✅ Модальное окно с формой  
✅ Бургер-меню на JS  

## Технологии
- HTML5 (БЭМ-нейминг)
- CSS3 (Flexbox, Grid, анимации)
- JavaScript (ES6, DOM API)
